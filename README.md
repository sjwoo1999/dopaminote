# 🧠 Dopaminote

도파민 소비 패턴을 분석하고 개선하는 디지털 웰빙 웹 애플리케이션입니다.

## 📋 서비스 개요

**Dopaminote**는 사용자의 도파민 소비 패턴을 분석하고 개선하는 디지털 웰빙 웹 애플리케이션입니다.

### 🎯 핵심 가치 제안
> "무의식적인 도파민 소비를 의식적으로 바꿔서 건강한 디지털 라이프를 만들어보세요"

---

## 🎯 문제 정의

### 현재 상황
- 스마트폰 사용 시간 증가
- 소셜미디어 중독 현상
- 무의식적인 스크롤링과 콘텐츠 소비
- 디지털 디톡스의 필요성 증가

### 해결하고자 하는 문제
- 도파민 소비 패턴의 무의식적 반복
- 디지털 웰빙에 대한 인식 부족
- 개인별 맞춤형 개선 방안 부재

---

## 🚀 서비스 기능

### 1. 📸 도파민 기록 (`/record`)
**기능**: 스크린샷 업로드와 상황/기분 기록
- **스크린샷 업로드**: 도파민을 소비하는 순간을 캡처
- **상황 선택**: 심심함, 스트레스, 습관, 소셜미디어, 업무, 엔터테인먼트, 기타
- **기분 기록**: 좋음, 무감정, 나쁨
- **추가 메모**: 상황에 대한 생각이나 느낌 기록

### 2. 📊 패턴 분석 (`/report`)
**기능**: 기록된 데이터 분석 및 시각화
- **통계 대시보드**: 총 기록 수, 평균 기분, 주요 상황
- **차트 시각화**: 상황별/기분별 분포 차트
- **개인화된 피드백**: 데이터 기반 개선 조언
- **패턴 발견**: 언제, 왜 도파민을 찾게 되는지 분석

### 3. 📝 회고 노트 (`/journal`)
**기능**: 일일 회고 작성 및 목표 설정
- **일일 회고**: 도파민 소비 패턴 돌아보기
- **목표 설정**: 개선을 위한 구체적 목표 수립
- **과거 기록 조회**: 회고 히스토리 확인
- **진행 상황 추적**: 목표 달성도 모니터링

## 👥 타겟 사용자

### 주요 타겟
- **디지털 웰빙에 관심 있는 20-30대**
- **스마트폰 사용 시간이 많은 직장인**
- **소셜미디어 중독을 느끼는 사용자**
- **자기계발에 관심 있는 개인**

### 사용자 페르소나
```
이름: 김민수 (28세, 회사원)
상황: 업무 중 스트레스로 인해 자주 소셜미디어를 확인
목표: 건강한 디지털 라이프를 만들고 싶음
동기: 무의식적인 스크롤링을 줄이고 의미 있는 시간을 만들고 싶음
```

## 🛠️ 기술적 특징

### 기술 스택
- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS 4
- **Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage (이미지 저장)
- **Charts**: Recharts
- **Icons**: Lucide React

### 아키텍처
- **App Router**: Next.js 15의 최신 라우팅 시스템
- **반응형 디자인**: 모바일/데스크톱 호환
- **타입 안정성**: TypeScript로 런타임 에러 방지
- **확장 가능성**: 컴포넌트 기반 모듈화 설계

## 🚀 시작하기

### 1. 저장소 클론

```bash
git clone <repository-url>
cd dopaminote
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 환경 변수 설정

프로젝트 루트에 `.env.local` 파일을 생성하고 다음 내용을 추가하세요:

```env
# Supabase 설정
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. 개발 서버 실행

```bash
npm run dev
```

개발 서버가 [http://localhost:3001](http://localhost:3001)에서 실행됩니다.
